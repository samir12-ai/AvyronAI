1. Transform the Current “AI Actions” Section into an Agent Control Box

The current "AI Actions" section is acting as a metrics log rather than an intelligent campaign manager. This must be redesigned into a compact AI Agent Box that functions as the operational brain of the dashboard.

The new component should be titled:

MarketMind Agent  
(or AI Campaign Manager)

This box must summarize the campaign situation and guide the user toward the next action rather than listing raw metrics.

The Agent Box must contain the following structured layers:

A. Campaign Status Summary  
A short high-level snapshot of the campaign state derived from system metrics and engines. This should summarize the situation in a way a user can understand within seconds.

Example information sources:
- CPA
- ROAS
- Content pipeline status
- Plan execution progress

Example display:

Campaign Status  
CPA: $10 | ROAS: 4x  
Content Progress: 1 / 50

This summary must not dump raw metrics, but synthesize them into readable status signals.

--------------------------------------------------

B. Strategic Insight Layer

The AI Agent must produce a short interpretation of the data coming from the engines and campaign metrics.

Example:

“Creative performance is strong (CTR indicates high resonance). Scaling winning creatives may reduce acquisition cost.”

This insight must be generated from the underlying system data, not hardcoded.

--------------------------------------------------

C. Priority Action Layer

The agent must always provide one clear recommended action.

Example:

Priority Action  
Create 2 reels today to maintain content cadence and reduce CPA.

This should be generated dynamically based on:

- Plan progress
- Content pipeline status
- Performance signals
- Engine recommendations

--------------------------------------------------

D. Action Buttons (Execution Shortcuts)

The Agent Box must include direct execution buttons connected to the system:

Examples:

Create Content  
Generate Content Idea  
Open Creation Studio

These buttons must route directly to the relevant modules.

--------------------------------------------------

E. Plan Progress Indicator

The agent must display plan execution progress clearly.

Example:

Content Plan Progress  
1 / 50 completed  
49 remaining

This information should come from the plan execution tracker.

--------------------------------------------------
