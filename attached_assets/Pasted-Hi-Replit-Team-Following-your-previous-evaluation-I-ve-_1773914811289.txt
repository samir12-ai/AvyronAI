## Updated Direction (Integrated, Not Layered)

### 1. SGL Upgrade (instead of separate Guards Layer)
- Add relevance-based ranking (confidence + semantic fit per engine/territory)
- Introduce evidence sufficiency rules (minimum signals per territory, no weak coverage)
- Remove global fallback (top-frequency signals)
- Keep purification as-is

Goal: improve input quality without over-restricting signal flow

---

### 2. CEL Upgrade → Semantic Enforcement (replace self-eval loop)
- Remove reliance on LLM self-scoring
- Introduce semantic alignment check (embedding-based, deterministic)
- Add:
  - drift detection (output vs mapped signals)
  - single repair pass (instead of retry loop)

Goal: enforce actual grounding, not declared traceability

---

### 3. SIV Upgrade → Semantic Validation (replace keyword overlap)
- Replace keyword matching with embedding similarity / concept-level comparison
- Eliminate false orphan detection caused by paraphrasing

Goal: reduce false warnings and improve true alignment detection

---

### 4. Claim Classification (new required component)
To support the balance between reasoning freedom and control:
- Classify output into:
  - factual claims (must be evidence-backed)
  - inferred insights
  - emotional synthesis
- This classifier is non-LLM-evaluative (rule + embedding or deterministic extraction)

Goal: prevent circular validation while preserving deep thinking + emotion

## Additional Gap Identified (Not Previously Addressed)

Even with grounding fixed, we are still observing:

- Strong outputs in one engine (e.g., Offer)
- Followed by generic or mismatched outputs in downstream engines (e.g., Persuasion)

### Root Cause:
Lack of structured carry-forward between engines

### Proposed Fix:

#### Engine-to-Engine Contract System
Each engine passes forward:
- Locked decisions (non-negotiable strategic elements)
- Structured transfer objects (why this output exists, required proof, constraints)
- Non-generic anchors (specific emotional tension, contrast, transformation language)

Goal: prevent abstraction drift and generic dilution across engines
