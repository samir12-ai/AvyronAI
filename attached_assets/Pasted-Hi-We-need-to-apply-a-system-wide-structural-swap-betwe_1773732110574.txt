We need to apply a system-wide structural swap between the Awareness Engine and the Funnel Engine.

This is not a cosmetic UI change and not a prompt tweak.  
It is an architecture-level execution-order correction that must be reflected across the entire pipeline.

## Current issue

Right now the system behaves as if:

Funnel → Awareness

which means the Funnel is selected first, and then Awareness analyzes or validates afterward.

This is creating contradictions such as:
- Awareness identifying proof-led entry
- Funnel still selecting challenge
- warnings appearing because the funnel type does not match the awareness route

In other words, the Awareness Engine is diagnosing correctly, but it is placed too late in the system to control downstream decisions.

## Required structural correction

We need to swap the execution order to:

Awareness → Funnel

This must be treated as a root-level pipeline change, not a local screen reorder.

## Important: apply this swap across the whole system

This is the most important part.

At the moment, the engines after Awareness are wired as if Awareness comes later in the chain.  
After the swap, the whole system must be re-bound around the new order.

That means:

- anything that currently waits for Funnel must be updated if it now needs Awareness first
- anything that currently consumes Awareness after Funnel must be re-bound to consume it before Funnel
- anything downstream that depends on funnel structure must now depend on the funnel generated from awareness, not on a pre-awareness funnel path

In other words:

### the entire dependency graph must be updated to reflect the new order

not just the two engine screens.

The new execution order should be:

Product DNA  
→ Market Intelligence  
→ Audience  
→ Positioning / Differentiation / Mechanism / Strategy Root  
→ Awareness Engine  
→ Funnel Engine  
→ all downstream engines that depend on funnel outputs

After this change:

- Funnel should not run without Awareness
- Downstream funnel-dependent engines should not run without Funnel
