🎯 Core Issue

The Narrative Layer is currently:
	•	Structurally correct ✅
	•	Deterministic and read-only ✅
	•	Properly placed post-engines ✅

But:
	•	It sometimes selects audience-level signals instead of market-level causal truth
	•	And more importantly:

⚠️ It can drift away from what the engine cards are actually showing

⸻

🚨 Critical Requirement (Most Important)

We need full alignment between:
	•	Narrative Layer (Strategic Narrative card)
	•	Engine Cards (Positioning, Differentiation, Mechanism, etc.)

The rule:

The Narrative MUST NOT introduce a different story than the engines.

If the engine cards say:
	•	Territory = X
	•	Mechanism = Y
	•	Root cause = Z

Then the Narrative must:
	•	Use the SAME logic
	•	Translate it (simplify it)
	•	But NEVER reinterpret or override it

⸻

🧠 Intended UX Behavior

We now have two layers:

1. Engine Cards
	•	Detailed
	•	Analytical
	•	For advanced users
	•	Source of truth

2. Narrative Layer
	•	Simplified
	•	Connected
	•	For clarity

👉 But both must describe the same system reality

Not:
	•	Engine says one thing
	•	Narrative says another

⸻
